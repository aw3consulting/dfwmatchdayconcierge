3. Compact horizontal navigation across the upper header.

## 8. Desktop Layout Behavior

Desktop layout target:

1. Full-width dark premium header bar.
2. Crest/logo zone on the far left.
3. Brand lockup adjacent to crest.
4. Navigation labels aligned horizontally across available space.
5. One primary action button at the far right.
6. Thin gold border or divider treatment.
7. Compact vertical height so the hero remains visually dominant.
8. Clear contrast and readable label spacing.

Desktop behavior must avoid:

1. Crowded text.
2. Navigation wrapping on desktop.
3. Overlarge logo that reduces hero impact.
4. Button dominance that makes the page feel like a sales form.
