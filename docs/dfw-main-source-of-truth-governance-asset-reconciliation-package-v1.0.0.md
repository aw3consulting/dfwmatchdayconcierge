## Purpose

This package records and governs the recovery from a catastrophic source-of-truth drift condition identified during the DFW Matchday Information + Planning Hub mockup program.

The drift condition is that high-value visual governance assets, package records, and approved mockup references were created or used while not all required governing assets were committed to and verified on `main`.

Because `main` is the governing source of truth, any high-value visual asset generation that depends on files outside verified `main` is not qualified.

## Catastrophic Drift Determination

The DFW homepage-linked index page mockup batch is the second highest ranking visual asset group after the approved homepage mockups.

Any operation involving this asset group requires full governance from `main`.

The system failed when it proceeded into high-value mockup generation without first verifying that all governing records and approved references were present on `main`.

This is not a minor branch synchronization issue. It is a source-of-truth failure.
