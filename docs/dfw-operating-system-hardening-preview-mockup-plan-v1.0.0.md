# DFW Operating System Hardening Preview Mockup Plan v1.0.0

## 1. Purpose

This file governs the hardening-preview mockup plan for the DFW Matchday Concierge Operating System UX.

The mockup plan converts `docs/dfw-operating-system-ux-blueprint-v1.0.0.md` into a controlled visual preview before any frontend code changes.

## 2. Control Boundary

This is a mockup-planning action only.

This action does not authorize:

- frontend code changes
- public claim expansion
- DreamHost production deployment
- official affiliation claims
- fake logo usage
- unsupported fulfillment claims
- live payment claims
- live dispatch claims

## 3. Source Blueprint

Source file:

`docs/dfw-operating-system-ux-blueprint-v1.0.0.md`

The mockup must follow:

- homepage funnel
- four-lane selector

- verified hub preview
- activated command hub
- VIP hub UX
- Team Family hub UX
- Media / Corporate hub UX
- Fan hub UX

- claim/status badges
- mobile hierarchy
- frontend implementation targets

## 4. Mockup Objective

The hardening-preview mockup must prove that the platform reads as a DFW World Cup Matchday Operating System.

It must not read as:

- generic transportation site
- black luxury landing page
- rideshare alternative only
- unsupported VIP service page
- ad-heavy fan blog
- fake official tournament site

## 6. Homepage Mockup Frame

Required above-the-fold elements:

- approved logo
- `DFW World Cup Matchday Operating System` positioning
- concise Dallas/Fort Worth welcome
- four-lane selector
- `Start Inquiry` CTA
- `Ask Concierge` CTA
- map/location preview
- operator-review language

Required lane cards:

1. VIP Guest Concierge
2. Team Family Support
3. Media / Corporate
4. Team Fan Hubs

Homepage must show that the visitor is building a plan, not simply booking a ride.

## 7. Four-Lane Selector Frame

Each lane card must show:

- lane name
- who it serves
- primary action
- claim/status badge
- confirmation boundary
- no unsupported guarantee

Lane card badge examples:

- VIP Guest Concierge: `Operator Review Required`
- Team Family Support: `Privacy-Sensitive`
- Media / Corporate: `Quote Required`
- Team Fan Hubs: `Free Guidance + Activated Planning`

## 8. AI Inquiry Frame

The AI inquiry frame must show:

- lane selection
- self vs coordinator question
- sensitive-status non-disclosure option
- arrival/departure fields
- hotel or area field
- matchday need chips
- food/cooler/luggage/security interest chips
- budget range
- communication preference
- submit for operator review

Required disclaimer:

`AI-assisted intake prepares a planning preview. Final availability, pricing, vehicle assignment, payment terms, and service confirmation require operator review.`

## 9. Verified Hub Preview Frame

The verified preview frame must show:

- secure access indicator
- intake summary
- missing details checklist
- recommended widgets
- preliminary itinerary sections
- activation CTA
- operator review required badge

Must not show:

- confirmed service
- final price
- confirmed Tahoe
- confirmed driver
- confirmed security
- official access confirmation

## 10. Activated Hub Frame

The activated hub frame must show:

- itinerary builder
- guest details
- quote status
- payment/deposit status
- operator-reviewed next steps
- service or referral routing
- status tracker

Required status states:

- Inquiry Received
- Intake Complete
- Operator Review Required
- Quote Pending
- Payment Pending
- Confirmed
- Assigned
- Service or Referral

## 11. VIP Hub Frame

The VIP frame must show:

- private header
- Black 2026 Chevrolet Tahoe standard
- Tahoe confirmation boundary
- VIP operator review status
- itinerary planner
- food, cooler, luggage, comfort, and security coordination request areas
- payment/deposit status
- privacy handling

Required copy boundary:

`Tahoe and operator assignment are confirmed only after operator review, quote, payment/deposit approval, and availability confirmation.`

## 12. Team Family Hub Frame

The Team Family frame must show:

- privacy-sensitive header
- representative/coordinator handling
- non-disclosure preference
- family/group itinerary
- VIP escalation review
- payment authority status
- operator review required badge

Ads are prohibited.

## 13. Media / Corporate Hub Frame

The Media / Corporate frame must show:

- organization/contact panel
- recurring schedule builder
- vehicle count request
- passenger/equipment count
- catering request
- company-card/payment pathway status
- quote required badge
- assignment status

Ads are prohibited inside private Media / Corporate hubs.

## 14. Team Fan Hub Frame

The fan frame must show:

- team selector
- language selector
- matchday card
- route planner
- Fan Experience panel

- food/cooler/sightseeing panels
- free guidance area

`Shared ride requests are operator-matched when timing, route, space, and guest fit make sense. Matching is not guaranteed until confirmed.`
