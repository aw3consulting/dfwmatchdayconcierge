The PDF-inspired homepage must be defined and anchored one section at a time.

1. Define section visual role.
2. Define allowed content posture.
3. Define blocked claims and functions.
4. Define desktop layout behavior.
5. Define mobile layout behavior.
6. Define component and token needs.
7. Present section for user review.
