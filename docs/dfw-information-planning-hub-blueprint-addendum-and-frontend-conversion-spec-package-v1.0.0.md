## 8. Command Hub Conversion Rule

The Information + Planning Hub must route visitors into the Command Hub system without implying confirmed service.

Correct conversion path:

`Public guide -> Inquiry -> Intake preview -> Operator review -> Hub preview -> Activation -> Quote / scope -> Payment or deposit status -> Confirmation -> Assignment -> Fulfillment -> Closeout`

The public site may show planning value.

The public site may not claim:

- confirmed booking before confirmation
- guaranteed driver assignment
- guaranteed concierge assignment
- guaranteed vehicle make or model
- guaranteed fulfillment
- live dispatch
- live routing
- live payment automation
- official tournament relationship
- official team relationship
- official venue relationship
- official sponsor relationship
