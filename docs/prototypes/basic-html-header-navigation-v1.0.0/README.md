# DFW Basic HTML Header Navigation v1.0.0

Status: BASIC STATIC HTML/CSS PROOF ARTIFACT

This prototype exists to prove basic HTML/CSS code-writing competence for a standard browser-rendered header/navigation unit.

It is not a homepage, not an approved header/navigation design, not a production route, and not a QR destination.

## Files

1. `index.html`
2. `assets/css/header.css`

## Scope

The artifact contains one semantic header, one logo image element, one brand lockup, one navigation element, eight navigation labels, and one placeholder button.
